# Options

|Option|Description|
|------|-----------|
| -l, - -lbin-auth <LBIN_AUTH>|Not required if you `export LBIN_AUTH=<token>`|
|  -f, - -file          |         Upload a file|
|  -i, - -std-input|              Command-line input to file upload|
|  -h, - -help    |               Print help|
|  -V, - -version|                Print version|
