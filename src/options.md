# Options

|Option|Description|
|------|-----------|
| -l, - -lbin-auth <LBIN_AUTH>|Not required if you `export LBIN_AUTH=<token>`|
|  -f, - -file          |         Upload a file|
|  -F, - -oneshot-file |          One-time use URL for an uploaded file|
|  -u, - -url         |           URL shortener|
|  -U, - -oneshot-url|            Make a one-time use URL|
|  -r, - -remote-url|             Link a file from a remote URL|
|  -i, - -std-input|              Command-line input to file upload|
|  -h, - -help    |               Print help|
|  -V, - -version|                Print version|
