 ## About
lbin is designed to interface exclusively with the file hosting service provided through [https://bin.liminal.cafe](https://bin.liminal.cafe).

### Compile
```
git clone https://codeberg.org/gnp/lbin.git
cd lbin
cargo build --release
```
