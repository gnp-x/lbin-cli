# lbin-cli

lbin-cli is designed to work exclusively with the file hosting service provided through [https://bin.liminal.cafe](https://bin.liminal.cafe), which is a pastebin [built by us](https://git.liminal.cafe/sakura/lbin).

## How
Read the [documentation](https://web.liminal.cafe/~sakura/lbin/).

## Compile
```
git clone https://git.liminal.cafe/sakura/lbin-cli.git
cd lbin-cli
cargo build --release
```
