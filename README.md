# lbin-cli

lbin-cli is designed to work exclusively with the file hosting service provided through [https://bin.liminal.cafe](https://bin.liminal.cafe), which is a pastebin [built by us](https://codeberg.org/gnp/lbin).

If you have any questions or have suggestions for lbin-cli, please create an [issue](https://codeberg.org/gnp/lbin-cli/issues).

## Compile
```
git clone https://git.liminal.cafe/sakura/lbin-cli.git
cd lbin-cli
cargo build --release
```
